# Pong_Game
Pong Game using SDL2  in C.
# Setup 
```
$ sudo apt install libsdl2-dev
$ sudo apt install libsdl2-ttf-dev
$ gcc pong.c `sdl2-config --cflags --libs` -lSDL2_ttf -o pong
$ ./pong
use w/s to control paddle .
```
